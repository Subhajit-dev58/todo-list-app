# 📝 Todo List App

A modern Todo List web application built using HTML, CSS, and JavaScript.

## 🚀 Live Demo
https://subhajit.github.io/todo-list-app/

## 📌 Features
- Add tasks
- Edit tasks (double click)
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Tasks saved using LocalStorage
- Responsive design
- Smooth animations

## 🛠️ Built With
- HTML
- CSS
- JavaScript
